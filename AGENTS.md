# Prototype Instructions

Run the local server yourself and open the preview in the in-app browser. Do not give the user server-start instructions when you can run it.

Before making substantial visual changes, use the Product Design plugin's `get-context` skill when the visual source is unclear or no longer matches the current goal. When the user gives durable prototype-specific design feedback, preferences, or decisions, record them in `AGENTS.md`.

When implementing from a selected generated mock, treat that image as the source of truth for layout, component anatomy, density, spacing, color, typography, visible content, and hierarchy.

## Locked Product Direction

- Mobile-first World Cup football-star personality quiz.
- Visual direction: premium collectible player card with oversized Chinese editorial headlines.
- Palette: charcoal black, electric lime, cobalt blue, white, restrained warm yellow.
- Result hierarchy: personality archetype first, football star second.
- The star match describes one on-pitch trait only, not the player's complete personality.
- Do not use official tournament logos, team crests, player portraits, or official card-brand layouts.
- V1.1 flow: landing page, 12 questions, four independently scored dimensions, 8 archetypes with subtypes, downloadable PNG result card.
- Dimension pairs: proactive attack / observation, individual leadership / collaboration, rational planning / intuitive creation, stable control / pressure burst.
